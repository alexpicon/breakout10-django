About this project

CPSC 5240, Breakout 10, Django MVC demo

Home About Admin

Small Django app I put together to show how the Model / View / Template pattern (Django's version of MVC) works.

How the layers map

  • Model: main/models.py has a Message class with one text field, saved to SQLite.
  • View: main/views.py has the request handlers. home lists every message, about renders this page.
  • Template: main/templates/main/about.html is this file. The about view renders it.
  • URL routing: mysite/urls.py hooks each path up to a view.
